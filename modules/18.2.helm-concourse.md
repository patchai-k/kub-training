    ```console
    helm install stable/concourse --version 5.1.2 --namespace ci --name concourse

1. Now lets connect to our Concourse instance

    >Note: this command uses the `ci` namespace

    ```
    kubectl port-forward -n ci $(kubectl get pods -n ci --selector=app=concourse-web --output=jsonpath="{.items..metadata.name}")  8080
    ```

    Now open the web preview on port 8080 to see the Concourse Dashboard.

1. Lets list our releases, and then delete Concourse

    ```
    helm list
    ```

    Output:

    ```
    NAME            REVISION        UPDATED                         STATUS          CHART           APP VERSION     NAMESPACE
    concourse       1               Thu Aug 22 16:01:25 2019        DEPLOYED        concourse-5.1.2 5.0.1           ci
    ```

    Lets cleanup our release:

    ```
    helm delete concourse
    ```

    Output:

    ```

    [Namespace] concourse-main
